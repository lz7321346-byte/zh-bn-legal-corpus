# 快速启动后端服务 - 完整步骤

## ⚠️ 重要：必须在项目目录中执行命令

## 步骤 1：进入项目目录

在 PowerShell 中执行：

```powershell
cd "C:\Users\周科伽、\Desktop\zh-bn-legal-corpus-main"
```

**验证是否在正确目录：**
```powershell
# 应该能看到 pyproject.toml 文件
ls pyproject.toml
```

## 步骤 2：创建虚拟环境（如果还没有）

```powershell
python -m venv .venv
```

## 步骤 3：激活虚拟环境

```powershell
.venv\Scripts\Activate.ps1
```

激活成功后，您会看到命令提示符前面有 `(.venv)` 标记。

## 步骤 4：安装依赖

**⚠️ 确保在项目目录中执行此命令！**

```powershell
# 先确认在正确目录
pwd
# 应该显示：C:\Users\周科伽、\Desktop\zh-bn-legal-corpus-main

# 然后安装依赖
pip install -e .[dev]
```

## 步骤 5：启动服务

```powershell
uvicorn backend.app.main:app --host 127.0.0.1 --port 8000 --reload
```

## 常见问题

### 问题：ERROR: file:///C:/Users/... does not appear to be a Python project

**原因：** 不在项目目录中执行命令

**解决：** 先执行 `cd "C:\Users\周科伽、\Desktop\zh-bn-legal-corpus-main"`

### 问题：uvicorn 命令找不到

**原因：** 依赖没有正确安装

**解决：** 
1. 确保在项目目录中
2. 确保虚拟环境已激活（看到 `(.venv)` 标记）
3. 重新执行 `pip install -e .[dev]`
