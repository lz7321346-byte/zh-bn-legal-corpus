# 启动后端服务 - 完整步骤（已修复）

## 步骤 1：进入项目目录

```powershell
cd "C:\Users\周科伽、\Desktop\zh-bn-legal-corpus-main"
```

## 步骤 2：创建虚拟环境

```powershell
python -m venv .venv
```

等待创建完成（可能需要一些时间）。

## 步骤 3：激活虚拟环境

```powershell
# PowerShell
& .\.venv\Scripts\Activate.ps1

# 或者使用完整路径
& "C:\Users\周科伽、\Desktop\zh-bn-legal-corpus-main\.venv\Scripts\Activate.ps1"
```

**如果遇到执行策略限制：**
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

激活成功后，命令提示符前面会显示 `(.venv)`。

## 步骤 4：升级 pip（可选但推荐）

```powershell
python -m pip install --upgrade pip
```

## 步骤 5：安装依赖

```powershell
pip install -e .[dev]
```

**注意：** 我已经修复了 `pyproject.toml`，现在应该可以正常安装了。

## 步骤 6：启动服务

```powershell
uvicorn backend.app.main:app --host 127.0.0.1 --port 8000 --reload
```

启动成功后，您应该看到：
```
INFO:     Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
INFO:     Started reloader process
INFO:     Started server process
INFO:     Waiting for application startup.
INFO:     Application startup complete.
```

然后可以：
- 访问 http://127.0.0.1:8000 查看服务信息
- 访问 http://127.0.0.1:8000/docs 查看 API 文档
- 访问 http://127.0.0.1:8000/api/v1/terms?q=宪法 测试搜索
